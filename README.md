# 10 Receitas que as Fábricas de Chocolate Escondem

**Autor:** Deassis Nascimento  
**Estilo:** Gourmet

Este projeto é um site em HTML estático contendo 10 receitas profissionais de chocolate que são verdadeiros segredos de fábrica. Ideal para confeiteiras, amantes de chocolate e empreendedores da confeitaria.

## **Acesse o site**

[https://seunome.github.io/site-receitas-chocolate](https://seunome.github.io/site-receitas-chocolate)

> O site é hospedado com HTTPS gratuito pelo GitHub Pages.

---

## **Receitas incluídas**

1. Ganache com Glucose e Manteiga  
2. Recheio Aerado com Leite em Pó  
3. Praliné Profissional  
4. Recheio de Trufa com Sorbitol  
5. Chocolate com Recheio que Derrete na Boca  
6. Temperagem com Mycryo  
7. Bombom com Duas Camadas de Recheio  
8. Bombons com Efeito Metálico  
9. Fudge Profissional com Marshmallow  
10. Recheio com Frutas Liofilizadas
